QR Generator  :::: 
🔗 Live Demo: https://aditi-adri.github.io/QR_generator/


🧭 How to Use

Open qr.html in any modern browser (Chrome, Edge, Firefox — no server required)

Set up your Profile (required for history).

Select a QR type: Text, URL, WiFi, Email, etc.

Fill in the required fields.

(Optional) Customize:

Size

Error Correction

Color

Click Generate QR Code : (or press Enter in single-line inputs)

Download or share your QR code: PNG / JPG / SVG / PDF download

Copy to clipboard

Share via WhatsApp, Email, or device sharing

All generated QR codes appear automatically in History.
